# Design System: Editorial Contemporâneo (Portfólio Digital)
**Project:** Robbi Zaidan Ahmad - Digital Portfolio
**Era:** 2026+ Jornalismo Visual

## 1. Visual Theme & Atmosphere
Artistic, Content-Rich, and Immersive. This design adopts a high-end digital culture and art magazine aesthetic (e.g., *The New Yorker*, *Vogue*, *Kinfold*). It uses strong vertical grid lines, thin double borders, high-contrast layouts, and elegant serif headlines to create a sophisticated, curated broadsheet appearance.

## 2. Color Palette & Roles
- **Primary Page Background:** Blanco/White (`#FFFFFF`) — A clean paper-like canvas for content.
- **Primary Text:** Preto/Black (`#000000`) — High readability, echoing traditional print media.
- **Accent Red:** Vermelho Vinho/Wine Red (`#800000`) — Employs premium contrast for hero actions, drop caps, section lines, and active states.
- **Dark Grey:** Cinza Escuro (`#333333`) — Used for subtle text components and captions.
- **Secondary Accents:**
  - Dourado/Gold (`#FFD700`) — Hover highlight accents, delicate borders, and visual ornaments.
  - Azul Marinho/Navy Blue (`#000080`) — Subtle focus elements.
  - Verde Esmeralda/Emerald (`#2ECC40`) — Successful validation alerts.
  - Bege/Beige (`#F5F5DC`) — Card backgrounds, secondary tabs, and paper-like containers.

## 3. Typography Rules
- **Display & Headings:** `Playfair Display, serif` — Decorative serif with distinct character shapes. Bold weights are used for sections and titles to reference editorial headlines.
- **Body Text:** `Lato, sans-serif` — Extremely clean, readable geometric sans-serif. Spaced with `line-height: 1.7` (`leading-relaxed`) to prevent reading fatigue.
- **Navigation & Labels:** Uppercase sans-serif with slight tracking/letter-spacing for an elegant, modern magazine look.

## 4. Component Stylings
- **Buttons:** Clean rects with sharp corners or very subtle rounding (`rounded-sm`). Transitions are animated with `all 0.5s ease-in-out` for a smooth page-turn or fade-in effect. Primary button uses Wine Red (`#800000`) or Solid Black (`#000000`) with high-contrast text and a Gold hover border.
- **Cards/Containers:** Styled like magazine column blocks. They feature thin solid borders (`1px solid #333333` or `#E0E0E0`) instead of heavy drop shadows, occasionally using beige (`#F5F5DC`) backgrounds to denote special segments.
- **Inputs/Forms:** Flat, minimal borders. Bottom border focus lines that scale or colorize from dark grey to wine red.
- **Decors:** Thin horizontal rules, uppercase subheaders, overlapping visual layers, and a Drop Cap on introductory text paragraphs.

## 5. Layout Principles
- **Grid Discipline:** Large clear lines (`border-bottom: 1px solid #E0E0E0`) separate key sections.
- **Asymmetric Columns:** Overlapping elements, combining text boxes that visually float on top of/beside images to replicate high-end editorial layouts.
- **Magazine Feel:** Micro-interactions (scale, border expand) on cards and tabs to simulate page-flipping and review.
